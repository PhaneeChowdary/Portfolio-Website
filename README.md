# Portfolio-Website

#### A personal portfolio website to showcase my skills and achievments.


## Installation

Follow below steps to run this project in your PC :    

- Download the project and open it in code editor.
- Change current working directory to this project folder.
- Now run following commands in terminal.

```bash
  mkdir virtualenv venv
  venv\Scripts\activate

  pip install django
  cd Mega

  py manage.py runserver
```
