# Portfolio-Website

<h1>To run this project on your local machine follow below steps : </h1>
<ol>
  <li>Clone or Download the Zipped project.</li>
  <li>Move to project directory.</li>
  <li>Make sure the version of python you are using is 3.8 or above.</li>
  <li>Open your commant prompt or terminal and type <strong>pip install -r requirements.txt</strong></li>
  <li>At last type <strong>python manage.py runserver</strong></li>
  <li>And follow the link that is provided by your machine to view on browser.</li>
</ol>
